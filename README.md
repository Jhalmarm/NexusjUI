# NexusjUI
Futuristic drop-in UI kit — floating nav, glass panels, 3D page transitions, HUD audio ticks, and a floating video player. Sci-fi interfaces that look like 2060. Pure HTML/CSS/JS. Kit de UI futurista drop-in: menús flotantes, paneles glass, transiciones 3D, audio HUD al tipear y video flotante. Interfaces tipo película 2060. HTML/CSS/JS puro.
